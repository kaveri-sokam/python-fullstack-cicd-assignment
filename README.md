# Python Full Stack CI/CD Starter Project

This starter project is intentionally incomplete.

Students must create:
- Jenkinsfile
- backend/Dockerfile
- frontend/Dockerfile
- docker-compose.yml
- sonar-project.properties

Stack
- Backend: FastAPI
- Frontend: React
- Database: PostgreSQL

Run backend:
pip install -r backend/requirements.txt
uvicorn main:app --reload
