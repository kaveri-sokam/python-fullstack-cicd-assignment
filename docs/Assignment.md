# Complete CI/CD Assignment

## Objective
Build an end-to-end CI/CD pipeline using Jenkins.

## You are provided
- Backend source code
- Frontend starter
- Tests

## You must create
1. Jenkins Installation
2. Freestyle Job
3. Declarative Pipeline (Jenkinsfile)
4. Scripted Pipeline
5. GitHub Webhook
6. SonarQube Integration
7. Backend Dockerfile
8. Frontend Dockerfile
9. docker-compose.yml
10. Smoke Tests

## Pipeline Stages
- Checkout
- Install Dependencies
- Backend Tests
- Frontend Build
- SonarQube Scan
- Build Docker Images
- Deploy with Docker Compose
- Smoke Test
- Cleanup

## Deliverables
- GitHub repository
- Jenkinsfile
- Dockerfiles
- docker-compose.yml
- SonarQube report
- Screenshots
- README

## Evaluation (100 Marks)
- Jenkins Setup: 10
- Freestyle Job: 10
- Declarative Pipeline: 20
- Scripted Pipeline: 15
- GitHub Integration: 10
- SonarQube: 10
- Docker: 15
- Deployment: 5
- Documentation: 5
